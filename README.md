# HR-Performance-Attendance-Dashboard
Objective
To analyze employee productivity and attendance for better HR decisions.
Designed a dynamic HR analytics dashboard using Advanced Excel, leveraging Pivot Tables, KPI metrics, and data visualization to evaluate employee performance, attendance trends, and operational efficiency.
Tools
-Excel
-Pivot Tables
-Charts
Features
-KPI Tracking
-Performance Analysis
-Attendance Monitoring
Insights
-High-performing employees identified
-Department-level comparison
-Attendance trends
